# Depthanalysis

Given a set of .pgn(.gz) files with engine evaluations, export the search 
depths to a .csv file.

By way of example, an analysis of the engines' search depth on book exit
for all the 1-threaded games from the [Fishtest LTC collection](https://huggingface.co/datasets/official-stockfish/fishtest_pgns) is shown in
the plots below.
